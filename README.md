# genten-web

Marketing and waitlist landing page for [Genten](https://genten.io) — a self-hosted,
real-time monitoring dashboard for AI agents.

## What this is

Static HTML/CSS landing page. No framework, no build step, no dependencies.
Just open `index.html` in a browser or deploy to any static host.

## Stack

- HTML
- CSS
- Vanilla JS

## Deployment

Deployed on Netlify. Every push to `main` auto-deploys to production.

## Local development

```bash
# Just open the file
open index.html

# Or use a local server
npx serve .
```
